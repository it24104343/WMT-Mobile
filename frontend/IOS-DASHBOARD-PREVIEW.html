📱 iOS Home Screen Dashboard

A fresh, modern app-grid style layout inspired by iOS home screen design patterns.

Features:

  • Grid-based app icon layout (4 columns)
  • Notification badges on app icons
  • Fixed iOS-style dock at the bottom
  • Glassmorphic design with backdrop blur
  • Status bar with time and indicators
  • Search bar for quick navigation
  • Smooth hover and tap animations
  • Fully responsive design
  • Touch-friendly spacing and sizing
  • Modern gradient backgrounds

How to Use:

  1. Import IOSDashboard component from ./components/IOSDashboard
  2. Use the theme "📱 iOS Home Screen" from the theme switcher
  3. Navigate to /ios-dashboard to view the app grid
  4. Customize app icons and labels in IOSDashboard.jsx
  5. Modify colors in theme-ios-dashboard.css

Customization:

Edit the apps array in IOSDashboard.jsx to add/remove app icons, change labels, icons, and links.
